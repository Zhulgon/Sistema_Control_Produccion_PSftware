# Patrones funcionales MES

Esta carpeta contiene los 12 modulos por patron usados como evidencia del proyecto MES funcional.

Los archivos originales `*_m.py` se conservaron como base historica de las actividades semanales. Para la fase final, `mes_functional_app.py` importa desde esta carpeta, de modo que la evidencia del proyecto queda separada y alineada con los UML funcionales.

Cada modulo incluye:

- Clases reales del patron.
- Funcion `run_demo()` con una demostracion pequena aplicada al MES.
- Compatibilidad con el flujo principal de `FunctionalMESProject`.
